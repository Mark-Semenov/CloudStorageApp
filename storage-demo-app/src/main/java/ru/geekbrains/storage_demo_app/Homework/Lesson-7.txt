Lesson 7

Сделал синхронизацию в следующем формате:
Когда пользователь заходит первый раз, создается root папка.
Далее пользователь может создавть папки, подпапки, что отображается слева в виде дерева (скриншот прилагаю).
Все папки привязываеются к root папке.
Если пользователь записал файлы, выбрав определенную папку, то эти файлы будут отображаться в таблице справа только тогда, когда эта папка выбрана.
Если выбранная папка пустая, то файлы отображаться не будут.

Если пользователь будет записывать файлы не создавая папки, то все файлы будут привязаны к root папке и будут отображаться в таблице при авторизации.
