Lesson 4-5

На текущий момент реализован практически весь базовый функционал:
1. Можно загрузить и выгрузить файлы
2. Запись файлов ведется в БД и на диск
3. Можно удалить файлы (удаление происходит одновременно с базы и с диска)
3. Подключена функция авторизации, можно залогиниться\разлогиниться
4. Можно добавлять роли (user, admin и тд) и ограничивать доступ к ресурсам
5. Файлы отображаются в виде таблицы (Имя, Тип, Размер)
